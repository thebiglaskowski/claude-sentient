# Decisions Log

This file captures decisions made during development for traceability.

## Format

```markdown
### [CATEGORY-NNN] Brief description
- **Time:** HH:MM
- **Context:** What were you working on?
- **Decision:** What did you decide?
- **Alternatives Considered:** What else was evaluated?
- **Rationale:** Why this choice?
- **Implications:** What are the consequences?
- **Commit:** (added after commit)
```

## Categories

| Prefix | Category |
|--------|----------|
| TECH | Technical implementation |
| ARCH | Architecture |
| TRADE | Trade-off |
| SCOPE | Scope change |
| SEC | Security |
| PERF | Performance |
| FIX | Bug fix approach |
| DEFER | Deferred work |
