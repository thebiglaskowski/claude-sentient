# Project Instructions

> Generated by Claude Conductor 2.0

## Quick Start

```bash
# Run autonomous loop
/cc-loop "implement feature X"

# Review code
/cc-review

# Run quality checks
/cc-assess
```

## Available Commands

| Command | Purpose |
|---------|---------|
| `/cc-loop` | Autonomous development loop |
| `/cc-plan` | Create feature specification |
| `/cc-review` | Code review |
| `/cc-test` | Run tests with coverage |
| `/cc-secure` | Security audit |
| `/cc-commit` | Create well-formatted commit |

## Project Configuration

Configuration in `.claude/config.json` overrides platform defaults.

## Extensions

Add project-specific extensions in:
- `.claude/extensions/skills/` — Custom skills
- `.claude/extensions/gates/` — Custom quality gates
- `.claude/extensions/overrides/` — Platform overrides

## State

State is stored in `.claude/state/`:
- `loop.state.json` — Current loop state
- `session.state.json` — Session metadata

Rendered Markdown versions are auto-generated for human readability.
